This is a demo page to show my Chrome extension without needing to download and intall it.

Pulls user feedback data for Dotdash websites and displays it on the page.

Instructions

  • Click a URL below to emulate navigating to a site.
  • Make sure the Feedback Overlay is enabled for that site in the settings on the right (i.e. if you clicked on a URL for lifewire.com, check the Lifewire box in the settings).
  • Click the [+] icon in the Feedback Overlay that appears in the bottom right corner to expand the comments (warning: real comments with some NSFW language!).
  • Select a date range in the settings to see the Feedback Overlay dynamically update (requires enough data to see - try using the first link and set the From date to around 03/01/2019).
  • Click the 'Hide Overlay' option in the settings to toggle off/on the Feedback Overlay.
